# Как добавить новую работу на сайт

Эта инструкция для вас — чтобы добавлять новые картины в галерею
самостоятельно, без обращения к разработчику.

## Шаги

1. Положите оригинал изображения в папку `src/`.
2. Отдайте файл на конвертацию в WebP (или сделайте сами через squoosh.app):
   - `images/preview/ваш-slug.webp` — ~800px по большей стороне
   - `images/full/ваш-slug.webp` — ~1600px по большей стороне
   - Не обрезайте кадр — только вписывайте в размер с сохранением пропорций.
3. Откройте `data/works.js` и добавьте новую запись в массив `WORKS`, например:

```js
{
  slug: 'moon-cat-2',
  title: { ru: 'Кот и луна 2', en: 'Cat and moon 2', uk: 'Кіт і місяць 2' },
  tags: { ru: ['Фэнтези'], en: ['Fantasy'], uk: ['Фентезі'] },
  featured: false,
  dateAdded: '2026-08-01',
  w: 800,
  h: 1000,
  preview: 'images/preview/moon-cat-2.webp',
  full: 'images/full/moon-cat-2.webp'
}
```

4. Важно про `dateAdded` — это дата публикации на сайте, а не дата,
   когда вы создали саму картину. От неё зависит бейдж "NEW".
5. Сохраните файл — работа появится в галерее и (если `featured: true`)
   на главной странице.

## Теги

Старайтесь не плодить новые теги под каждую работу — используйте уже
существующий набор (Dark Art, Fantasy, Surreal, Cyberpunk, Cartoon/Animation
и т.п.). Если тегов станет слишком много, фильтр на сайте станет неудобным.
